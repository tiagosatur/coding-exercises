# Node.js Exercises

A collection of Node.js exercises focused on backend development, APIs, and data processing.

## Exercises

- [file-upload](./file-upload) — CSV inventory upload and analysis API using Express and Multer.
- [test-framework](./test-framework) — build a Jest-like test runner from scratch (`it`, `exec`, `isEqual`), sync then async.

## Exercise structure

Each exercise follows the same pattern:

```
<exercise-name>/
  starter/       ← blank starting point
  submissions/
    01/          ← first attempt
    02/          ← revised attempt
  README.md      ← requirements and instructions
  COURSE.md      ← step-by-step guide with explanations
```
