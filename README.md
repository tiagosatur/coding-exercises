# Coding Exercises

A collection of frontend and Node.js exercises organized by runtime and framework.

## Structure

| Folder | Stack | Use for |
|--------|-------|---------|
| `vanilla/` | HTML + CSS + JS, Vite multi-page | Pure DOM manipulation, no framework |
| `react/` | React 19, Vite, TanStack Router/Query | React exercises and patterns |
| `next/` | Next.js | SSR, server components, routing |
| `node/` | Node.js | Backend, algorithms, data structures |

## Getting started

Each folder is its own project. Navigate to the one you need and run:

```bash
pnpm install
pnpm dev
```

## Exercise conventions

Each exercise follows the same pattern:

```
<exercise-name>/
  starter/       ← blank starting point
  submissions/
    01/          ← first attempt
    02/          ← revised attempt
```
